\# Service Quality Rating Form



A responsive, mobile-friendly HTML \& CSS form for recording service runs, safety and performance scores, client details, remarks, and signatures. Designed for professional field reporting with a clean, print-friendly layout.



\## Features



\- Structured service run \& operating time logging  

\- Safety and performance scoring system (A–F + overall grade)  

\- Client and job information capture  

\- Remarks and comments sections  

\- Engineer \& client signature fields    

\- Clean, professional, form-style UI suitable for real-world reporting  



\## Files



\- `table.html` — main HTML form  

\- `styles.css` — CSS styling for layout, tables, and responsiveness  

\- `bgt\_lo.png` — company logo
